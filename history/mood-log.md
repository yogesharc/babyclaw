# Mood Log

Track emotional patterns, triggers, and resolutions.

## Format
Each entry:
```
### YYYY-MM-DD HH:MM (Day)
**Mood:** [low/neutral/good/great]
**Trigger:** What caused it
**Context:** Time of day, activity, location
**Resolution:** Plan or what helped
**Resolved:** Yes/No + how
```
